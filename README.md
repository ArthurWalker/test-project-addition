## Test Project Addition

This project implements programs to add together their command-line arguments in various programming languages.

### Build

Use:
```shell
$ make build
```

### Run

Java:
```shell
$ java Add 1 2 3
```
```
6
```

Python (2):
```shell
$ python add.py 1 2 3
```
```
6
```

### Tests

Use:
```shell
$ make test
```
